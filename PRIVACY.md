# AI Chat Reader 隐私政策

**最后更新：2026年3月7日**

## 概述
AI Chat Reader 是一款浏览器扩展，用于在 AI 聊天网站上快速浏览、定位和导出对话内容。我们重视用户隐私，本扩展不收集任何用户个人数据。

## 数据收集
本扩展**不收集、不存储、不传输**任何用户个人数据至外部服务器。

## 本地存储
扩展使用浏览器的 Chrome Storage API 在本地存储以下信息：
- 用户界面偏好设置（如悬浮球位置、侧边栏状态）
- 用户自行配置的 AI API 设置

这些数据仅保存在用户的本地浏览器中。

## 页面内容访问
扩展会读取 AI 聊天网站上的对话内容，仅用于：
- 在侧边栏中显示对话列表
- 导出对话为 Markdown 文件
- 当用户主动触发时，发送至用户配置的 AI 服务进行润色

扩展不会将页面内容上传至开发者的服务器。

## 第三方服务
当用户使用"AI 润色"功能时，选中的文本会发送至用户自行配置的第三方 AI 服务（如阿里云千问、OpenAI 等）。这些服务的数据处理受其各自隐私政策约束。

## 权限说明
- `storage`：用于保存用户设置偏好
- `host_permissions`：用于调用 AI API 实现润色功能
